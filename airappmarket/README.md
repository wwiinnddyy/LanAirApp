# AirAppMarket

`airappmarket/` is the official plugin market source for LanMountainDesktop, published from the `LanAirApp` repository.

Current scope:
- Host the official `index.json`
- Host the JSON schema
- Host static market assets
- Host the local and CI validator

Official index URL:
`https://raw.githubusercontent.com/wwiinnddyy/LanAirApp/main/airappmarket/index.json`

Notes:
- This repository only hosts the official source, not multi-source aggregation.
- The market UI is now built into LanMountainDesktop itself.
- Plugin packages referenced by the index should point to released `.laapp` files committed in this repository.
