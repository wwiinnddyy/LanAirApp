# LanMountainDesktop.SamplePlugin

## 中文

这是阑山桌面的标准示例插件，用于演示一套完整的插件开发与验证路径。

### 演示内容

- `plugin.json` 清单定义
- 插件入口类与生命周期
- 设置页注册
- 桌面组件注册
- 插件服务注册
- 本地化资源组织
- `.laapp` 打包与宿主安装

### 说明

- 这里的样例适合作为开发模板。
- 当前官方市场测试更推荐使用独立插件仓库版本。
- 如果你要创建自己的插件，建议先复制这个目录，再替换插件 ID、程序集名称和 README。

## English

This is the standard sample plugin for LanMountainDesktop.

### It demonstrates

- `plugin.json` manifest definition
- plugin entrance and lifecycle
- settings page registration
- desktop component registration
- service registration
- localization layout
- `.laapp` packaging and host installation
