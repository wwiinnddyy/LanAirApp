# 插件标准说明

## 中文

本目录存放阑山桌面插件开发需要遵循的基础约定和模板文件。

### 当前标准

- 安装包扩展名：`.laapp`
- 插件清单文件：`plugin.json`
- 本地化目录：`Localization/`
- 推荐语言文件：`zh-CN.json`、`en-US.json`
- 仓库根目录产物：`.laapp` 与 `README.md`

### 建议

- 中文作为主文档和主界面语言。
- 英文作为附加语言资源。
- 清单字段尽量保持稳定，避免频繁改动插件 ID 和入口程序集。

## English

This directory stores the baseline conventions and templates for LanMountainDesktop plugin development.

### Current standards

- Package extension: `.laapp`
- Manifest file: `plugin.json`
- Localization directory: `Localization/`
- Recommended languages: `zh-CN.json` and `en-US.json`
- Repository root deliverables: `.laapp` and `README.md`
